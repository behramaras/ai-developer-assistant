# AI Developer Assistant

A local AI assistant for developers that helps break down feature ideas into tasks, generates technical specifications, starter Python code, and lists potential edge cases. Built using [GPT4All](https://github.com/nomic-ai/gpt4all) with the **Meta LLaMA 3 8B Instruct Q4_0 model**.

---

## Features

- Breaks down feature ideas into clear development tasks
- Generates detailed technical specifications
- Produces starter Python code based on the specifications
- Lists potential edge cases for the feature
